# Azot Galam TV Extension

<img src="https://galam.sequoia-api.lfstrm.tv/upload/56b03295133a00b51ba10b3997ef5c82.svg" height="100" alt="GalamTV" />

This extension for [Azot](https://github.com/azot-labs/azot) lets you to download movies, series and shows from the [Galam TV](https://galamtv.kz/) streaming service.
